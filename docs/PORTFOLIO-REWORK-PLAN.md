# Reestructuración del portafolio Full Stack Mid+

Estado: en ejecución  
Rama: `feat/portfolio-rework-mid-plus`  
Sitio canónico previsto: Vercel  
Espejo: GitHub Pages

## Decisiones cerradas

- Posicionamiento: Full Stack Mid+.
- Casos principales: Adastra, CMS + Landing, Gateway + Datos y Hostlyc.
- Dirección visual: clara profesional; arquitecturas oscuras.
- Renderizado: Angular SSG.
- Idioma inicial: español, preparado para inglés.
- Regla editorial: no publicar métricas sin evidencia.

## Frentes de trabajo

- [ ] Contenido verificable y cuatro casos de estudio.
- [ ] Home reducido y sistema visual claro.
- [ ] Casos con arquitecturas semánticas específicas.
- [ ] Demos accesibles y realmente mobile-first.
- [ ] SSG, SEO por ruta, Vercel y espejo Pages.
- [ ] Pruebas unitarias, visuales, accesibilidad y rendimiento.

## Criterios de aceptación

- Los casos aparecen dentro de las primeras dos pantallas en 390 × 844.
- El home no supera aproximadamente siete pantallas móviles.
- No existe desbordamiento horizontal entre 360 y 1440 px.
- Texto normal mínimo de 16 px, metadatos de 12 px y controles de 44 px.
- Todas las rutas públicas responden con HTML prerenderizado.
- Lighthouse móvil: rendimiento 90, accesibilidad 95, buenas prácticas 95 y SEO 95.
- Ninguna cifra no verificada aparece publicada.
